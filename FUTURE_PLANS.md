# Future Plans

This file tracks ideas and concrete next steps for this repo so I can work
through them over time.

## Primary Goal
Introduce technical analysis tools based on Metodo Tendencia.

## Planned Work
- Define the Metodo Tendencia ruleset in plain language and convert it into
  programmable conditions.
- Create one foundational indicator that visualizes trend phases and signals.
- Add a companion strategy script for backtesting the Metodo Tendencia rules.
- Add alert conditions for entry/exit signals.
- Document usage examples and typical settings in a README section.

## Backlog Ideas
- Multi-timeframe confirmation version of the indicator.
- Risk management overlays (stops, targets, trailing rules).
- A simple dashboard/panel with current trend status and signal summary.

## Notes
- Keep scripts compatible with TradingView Pine Script (latest supported
  version).
- Prefer clear naming and comments so future edits are easy.
